# Foundation-of-data-analytics1-portfolio
Title of Project: Lucky Food Truck Analysis
Tools used: Google Sheets
Project Description:
This project, completed for a class, involved analyzing data from a Food Truck to help the owner understand factors affecting their business revenue. It consisted of weeks of analysis during which I cleaned data, created charts and pivot tables, and presented findings as actionable insights. In my final analysis, I provide recommendations to improve sales based on season, day of the week, and weather. I created three visually appealing charts that summarize my analysis to improve broader understanding. Throughout this project, I learned to use dashboards more effectively, created pivot tables, and identified outliers. In my future role as a data professional, I know this project will improve the quality of my work because it taught me to perform polished analysis and present actionable insights.
